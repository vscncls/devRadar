# DevRadar

Find developers close to you with similar interests. Developed on "Semana OmniStack".

### Features

- Express REST API
- React Frontend
- Mobile app with React Native and expo

## Build

```shell
git clone https://github.com/vscncls/devRadar.git
cd devRadar
```

cd into each folder and run

```shell
yarn
```

## Runing

```shell
cd backend
yarn dev

cd frontend
yarn start

cd mobile
yarn start
```
