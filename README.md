# DevRadar

Find developers close to you with similar interests.

### Features

- REST API
- React Frontend
- Mobile app with React Native and expo

## Build

```bash
git clone https://github.com/ramenbroth/devRadar.git
cd devRadar
```

cd into each folder and run

```bash
yarn
```

## Runing

```bash
cd backend
yarn start

cd frontend
yarn start

cd mobile
yarn start
```
